GOG Database

Welcome to the GOG Database, a third-party website that collects data on GOG games and tries to display it in a user-friendly manner. It's currently in beta state, because a lot of planned functionality is still missing.

To get started visit the products site and search for a game. Click on the name or ID to go to its info page.

If you found bugs or want to request a feature, you can contact me on freenode IRC (Yepoleb), Discord (Yepoleb#6459), Steam or GOG (slowest option because they still don't have a Linux client).
